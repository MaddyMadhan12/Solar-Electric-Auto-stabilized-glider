# Solar-Electric-Auto-stabilized-glider
---------------------------------------
This project is a lightweight solar RC glider built for long, efficient flight. Solar panels extend battery life, and a brushless motor provides takeoff power before gliding. Sensors and servos add basic auto-stability and control. An onboard camera captures real-time aerial footage.

Why I Started This Project
--------------------------
I’m passionate about aerospace engineering and long-endurance aircraft. Real solar UAVs and gliders can stay airborne for extremely long periods, and I wanted to build a scaled experimental prototype to understand the real challenges behind them.

Instead of just building a normal RC plane, I wanted to explore:

 -How solar panels can extend flight time
 
 -How autopilot systems stabilize aircraft
 
 -How onboard electronics work together in real flight

This project is meant to be a learning and experimentation platform.

Main Features
-------------
 -Lightweight fixed-wing glider airframe
 
 -Solar-assisted power system
 
 -Flight controller with autopilot stabilization
  
 -GPS and telemetry support

Project Goals
------------- 
 -Design and assemble a lightweight glider airframe
 
 -Integrate solar charging with Li-Po battery system
 
 -Configure flight controller for stable autonomous flight
 
 -Add onboard telemetry and video recording
 
 -Test and improve flight endurance
 
 -Onboard video recording (ESP32-CAM)
 
 -Long-endurance flight design focus
