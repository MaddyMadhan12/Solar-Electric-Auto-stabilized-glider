#COMPONENTS NEEDED / BILL OF MATERIALS
--------------------------------------
##Flight Controller & Avionics
------------------------------
Pixhawk Flight Controller (with GPS + compass module)
Power Module for Pixhawk
Micro SD card (for flight logs)
Buzzer (flight controller alarm)
Safety switch

##Telemetry & Radio Communication
---------------------------------
Telemetry Radio Set (433/915 MHz pair)
RC Transmitter + Receiver (minimum 6–9 channels)

##Propulsion System
-------------------
Brushless Motor (≈ 1400KV class)
30A Brushless ESC
Folding Propeller (with spinner)
Prop adapter / collet

##Power System (Main Battery + Regulation)
------------------------------------------
3S LiPo Battery (2200–3000 mAh)
LiPo Battery Charger
XT60 connectors (male & female)
Power distribution wires (14–22 AWG)
Heat shrink tubing

##Solar Charging System
-----------------------
Solar Panels (lightweight RC type)
CN3722 Solar Charger Module
3S LiPo BMS Protection Board
LM2596 Step-Down Voltage Regulator

##Onboard Vision System
-----------------------
ESP32-CAM Module
FTDI Programmer (for uploading code)

##Airframe & 3D Printing 
------------------------
Apex Glider STL Files
PLA or LW-PLA Filament
Carbon Fiber Tubes (wing spar reinforcement)
Control horns
Pushrods
Clevis links
Hinges
Screws and fasteners set

##Servos & Wiring
-----------------
4–6 Micro Servos (9g class)
Servo extension wires
Y-connectors (if required)

##Assembly & Electronics Tools
------------------------------
Soldering iron + solder
Wire stripper
Multimeter
Screwdriver set
Hex key set
CA glue + activator
Epoxy
Hot glue gun
Sandpaper
Hobby knife
